## Домашнее задание 1 -- Homework 1
1. Выберите, пожалуйста, набор данных для проекта. Он должен
содержать как минимум один групповой признак, несколько
непрерывных признаков. Коротко опишите предметную область.
2. Представьте численное описание данных (среднее, медиану,
минимальное и максимальное значения по групповому признаку; см.
команды base::summary() и dplyr::summarise()).
3. Представьте график «ящик с усами» с группами, отвечающими
различному полу (подсказку по коду можно найти в галерее
графических возможностей R). Это задание можно выполнить при
помощи базовой графики R или же с использованием пакета ggplot2.
4. Приведите графическое представление основных переменных
выбранного датасета, для вдохновения можно использовать
https://r-graph-gallery.com/.
5. (опционально) Для отчета используйте, пожалуйста, Markdown файл
(тут лежат подсказки, как пользоваться RMarkdown).

to different genders (a hint on the code can be found in the gallery
